Radar Chart


                        

                    

    
                    

    
                        

                        
                            
                        
                            
    
                        

    
                        


                            

                                
                                A Radar Chart is a way of showing multiple data points and the variation between them.
                                
                                
In this example of Radar Chart, there are 3 Datasets :
                            
                            


                            
    

                                
  • 
                                    Comments,
                                
    • 

                                
  • 
                                    Duplicated Lines,
                                
    • 

                                
  • 
                                    Total Lines,
                                
    • 

                            


                            


                                and 5 different Versions of a software.

                                
Each dataset contains a number of values for these 5 versions.

                            

                            
                        

    
                    

    
                






    
                


                    


                        
Bubble Chart


                        

                    


                    

    
                        

                        
                            
                        
                            
    
                        

    
                        


                            

                                
                                A Bubble Chart is used to display three dimensions of data at the same time.
                                
                                
In this example of Bubble Chart, there are 2 Datasets :
                            
                            


                            
    

                                
  • 
                                    Total Lines,
                                
    • 

                                
  • 
                                    Comments,
                                
    • 

                            


                            


                                and 5 different Versions of a software.

                                
Each dataset is described by 3 dimensions, the corresponding horizontal and vertical axes (Versions and Dataset Data), and the third dimension which is the size of the individual bubbles and represents the Comments Rate.

                            

                            
                        

    
                    

    
                

    
                


                    


                        
Doughnut Chart


                        

                    

    
                    

    
                        

                        
                            
                        
                            
    
                        

    
                        


                            

                                
                                A Doughnut Chart is divided into segments, the arc of each segment shows the proportional value of each piece of data.
                                
                                
In this example of Doughnut Chart, there are 3 Datasets :
                            


                            
    

                                
  • 
                                    NOF,
                                
    • 

                                
  • 
                                    NOA,
                                
    • 

                                
  • 
                                    NOM,
                                
    • 

                            


                            


                                and 3 different Versions of a software.
                                
                                
Each Version contains a number of segments, and every one of them is equal to the number of data of each dataset.

Polar Area Chart


                        

                    

    
                    

    
                        

    
                            
                            
                            

                        

    
                        


                            

                                
                                A Polar Area Chart is similar to a Pie Chart, with the only difference that each segment has the same angle but the radius is depending on the value.
                                
                                
In this example of Polar Area Chart, there are 2 Datasets :
                            
                            


                            
    

                                
  • 
                                    Duplicated Lines,
                                
    • 

                                
  • 
                                    Duplicated Blocks,
                                
    • 

                            


                            

                                
                                and 3 different Versions of a software.
                                
                                
Each Version contains a number of segments, and every one of them is equal to the number of data of each dataset.

                            

                            
                        

    
                    

    
                

    
                
    
                


                    


                        
Bar Chart


                        

                    

        
                    


                        

                            
                            
                            
                            
        
                        


                        


                            

                                
                                A Bar Chart allows a number of properties to be specified for each dataset. These are used to set display properties for a specific dataset.
                                
                                
In this example of Bar Chart, there are 3 Datasets :
                            
                            


                            
    

                                
  • 
                                    Commits,
                                
    • 

                                
  • 
                                    Bugs,
                                
    • 

                                
  • 
                                    Issues,
                                
    • 

                            


                            

                                
                                and 5 different Versions of a software.
                                
                                
Each Version contains a number of different bars, which is equal to the total number of datasets.
                            
                            

                            
                        


                    


                    
    
                

    
                


                    


                        
Line Chart


                        

                    

    
                    

    
                        

                        
                            
                        
                            
    
                        

    
                        


                            

                                
                                A Line Chart is used for plotting data points on a line.
                                
                                
In this example of Line Chart, there are 3 Datasets :
                            
                            


                            
    

                                
  • 
                                    Analyzed Files,
                                
    • 

                                
  • 
                                    Directories,
                                
    • 

                                
  • 
                                    Bugs,
                                
    • 

                            


                            

                                
                                and 5 different Versions of a software.
                                
                                
For each Version there is a point that is shown in the position of the number of data of each dataset.
